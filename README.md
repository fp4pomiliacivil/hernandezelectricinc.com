# Hernandez Electric Website

Static website for **Hernandez Electric Inc.** at [hernandezelectricinc.com](https://hernandezelectricinc.com/).

The production site files live in `netlify-export/`.

## Pages

- `netlify-export/index.html` — Home
- `netlify-export/about/index.html` — About Us
- `netlify-export/services/index.html` — Our Services
- `netlify-export/contact/index.html` — Contact Us

## Preview Locally

```bash
cd hernandez-electric-website
python3 -m http.server 3457 --directory netlify-export
```

Open [http://localhost:3457](http://localhost:3457).

## Deploy

Upload the contents of `netlify-export/` to Netlify for `hernandezelectricinc.com`.
